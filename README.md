#Simplechaindb

based on BigchainDB

## 初次安装部署启动
```
git clone https://git.oschina.net/buaalining/Simplechaindb.git
cd Simplechaindb/clusterdeploy
vim blockchain-node
./first_setup.sh $NUM_NODES
```

## 更新部署启动
```
cd Simplechaindb
git pull
cd clusterdeploy
./update $NUM_NODES
```
